・はじめに

　Windows 8および8.1のロック画面の有効無効をクリックだけで切り替えることができる
ソフトウェアです。
　レジストリをいじりますので、嫌な方はご使用をお控えください。


・ファイル構成

　-NoLockScreen.exe
　-readme.exe(このファイル)
　
・使い方
　
　NoLockScreen.exeをダブルクリックします。
　ユーザーアカウント制御ダイアログが表示されるので、「はい」をクリックします。
　「ロック画面を無効化」ボタンをクリックします。
　「ロック画面を無効化しました」とダイアログが表示されたら成功です。「OK」をクリックします。
　一度Windowsからサインアウトしてください。
　これで、ロック画面は無効化されました。
　もう一度同じ手順を踏むと、ロック画面は有効化されます。
